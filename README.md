# Peek-a-Login

A playful login page where cute monsters watch your every move. Their eyes follow your cursor, they lean in when you focus on the form, and they politely look away when you type your password.

**[Live Demo](https://peek-a-login.vercel.app/login)**

## Demo
![peek-a-login-demo](https://github.com/user-attachments/assets/6afd0e10-8850-490e-9784-7ec3ff7e5883)

## Features

- **Eye tracking** - Monsters follow your cursor with smooth, spring-based animations
- **Form reactions** - They lean forward curiously when you click on input fields
- **Privacy mode** - Toggle password visibility and watch them avert their gaze

## Getting Started

```bash
# Install dependencies
bun install

# Start dev server
bun dev
```

Open [http://localhost:5173](http://localhost:5173) to see it in action.

## Tech Stack

React, TypeScript, Vite, Tailwind CSS, Framer Motion
